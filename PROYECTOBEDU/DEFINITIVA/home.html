Durante esta etapa del Proyecto se realizaron diversos modelos de series de tiempo (ARIMAX) para poder llevar a cabo pronósticos y poder entender el tiempo que tardará en recuperarse el empleo de nuestro país. Leer a detalle metodología de proyecto

Sectores económicos


Se realizó un análisis desagregado por sectores económicos para poder entender el impacto de la pandemia en cada uno de estos:


Sector primario


Para este caso el modelo ARIMA utilizado fue uno de orden: (1,0,2). Respecto a la gráfica resultante con los pronósticos para 2021 podemos observar que el empleo en este sector tiene un comportamiento muy volátil y esto se aprecia en el pronóstico realizado.


Sector secundario

Para este caso el modelo ARIMA utilizado fue uno de orden: (4,1,0). Respecto a la gráfica resultante con los pronósticos para 2021 podemos observar que el empleo en este sector ha tenido una tendencia positiva con una gran caída los para los meses que inició la pandemia, de igual forma podemos observar que para este año no se recuperará por completo el empleo.


Sector terciario

Para este caso el modelo ARIMA utilizado fue uno de orden: (2,1,3). Respecto a la gráfica resultante con los pronósticos para 2021 podemos observar que el empleo en este sector tiene un comportamiento con tendencia positiva, pero podemos ver que para este año 2021, el empleo en este sector estará lejos de recuperarse.


Nivel de ingreso

Se llevó a cabo un análisis desagregado por nivel de ingreso para poder identificar cual fue el nivel de ingreso más afectado por la pandemia. Menos de un salario mínimo Para este caso el modelo ARIMA utilizado fue uno de orden: (0,1,1). Respecto a la gráfica resultante con los pronósticos para 2021 podemos observar que el empleo en este nivel de ingreso tiene un comportamiento muy acelerado con una tendencia positiva y con un alza a partir de que comenzó la pandemia, esto posiblemente se explique ya que, ante la falta de trabajo mejor pagado, mucha gente disminuyó su nivel de ingreso.

Entre uno y dos salarios mínimos


Para este caso el modelo ARIMA utilizado fue uno de orden: (2,1,0). Respecto a la gráfica resultante con los pronósticos para 2021 podemos observar que el empleo en este nivel de ingreso al igual que en el anterior tiene un comportamiento muy acelerado con una tendencia positiva y con un alza a partir de que comenzó la pandemia, esto posiblemente se explique ya que, ante la falta de trabajo mejor pagado, mucha gente disminuyó su nivel de ingreso.


Entre dos y tres salarios mínimos


Para este caso el modelo ARIMA utilizado fue uno de orden: (0,1,1). Respecto a la gráfica resultante con los pronósticos para 2021 podemos observar que el empleo en este nivel de ingreso tiene un comportamiento cambiante con una gran caída cuando comenzó la pandemia, que no ha podido ni podrá recuperarse este año.


Entre tres y cinco salarios mínimos


Para este caso el modelo ARIMA utilizado fue uno de orden: (0,1,1). Respecto a la gráfica resultante con los pronósticos para 2021 podemos observar que el empleo en este nivel de ingreso tiene un comportamiento a la baja, con una gran caída desde que inició la pandemia la cual parece que no ha acabado y se estabilizará este año, pero probablemente ya no se pueda recuperar ese empleo a su nivel previo.


Más de cinco salarios mínimos

Para este caso el modelo ARIMA utilizado fue uno de orden: (0,1,1). Respecto a la gráfica resultante con los pronósticos para 2021 podemos observar que el empleo en este nivel de ingreso tiene un comportamiento a la baja que parece no haber sido muy afectado por la pandemia y posiblemente este comportamiento se deba a causas más de índole estructural de nuestra economía.


Tamaño de la unidad económica


Se llevó a cabo un análisis desagregado por tamaño de la unidad económica para poder identificar cuales fueron las empresas más afectadas durante esta pandemia.


Micronegocios


Para este caso el modelo ARIMA utilizado fue uno de orden: (0,1,1). Respecto a la gráfica resultante con los pronósticos para 2021 podemos observar que el empleo en estas unidades económicas tuvo una gran caída para el momento inicial de la pandemia, pero tuvo una recuperación muy rápida, y se prevé que este año prácticamente regrese a sus niveles antes de la pandemia.


Pequeños negocios


Para este caso el modelo ARIMA utilizado fue uno de orden: (2,1,2). Respecto a la gráfica resultante con los pronósticos para 2021 podemos observar que el empleo en estas unidades económicas tuvo una gran caída para el momento inicial de la pandemia, pero tuvo una recuperación muy rápida, aunque parece que va a tardar varios años en recuperar el nivel que se tenía previo a la pandemia.


Medianos negocios


Para este caso el modelo ARIMA utilizado fue uno de orden: (2,1,2). Respecto a la gráfica resultante con los pronósticos para 2021 podemos observar que el empleo en estas unidades económicas tuvo una caída moderada para el momento que comenzó la pandemia, aunque ha tenido y se prevé que tenga una recuperación muy lenta.


Grandes negocios


Para este caso el modelo ARIMA utilizado fue uno de orden: (2,1,2). Respecto a la gráfica resultante con los pronósticos para 2021 podemos observar que el empleo en estas unidades económicas tuvo una gran caída para el momento que comenzó la pandemia, aunque ha tenido y se prevé que tenga una recuperación lenta, tardando en llegar a su nivel previo.
